# v2.0 优化研究 — 交叉分析与方案收敛

> **日期**: 2026-06-03
> **基于**: 7维度系统研究（D1-D7）
> **目的**: 识别优化间的冲突与协同，给出最终方案收敛

## 一、冲突矩阵

共识别出 **8 对跨维度交互**，其中 **2 对需要协调，6 对兼容**。

### 1.1 严重冲突（需取舍）

| # | 维度对 | 冲突描述 | 严重度 |
|---|--------|---------|:------:|
| C1 | D1(因子增至6个) vs D5(持股减至8只) | 因子越多→选股越分散→需要更多持股来承载因子暴露。6因子+8只股票导致每只股票平均承担0.75个因子，分散不足 | ⚠️ 中 |
| C2 | D2(IC加权) vs D1(等权哲学) | D2研究推荐IC加权（+2-4%年化），D1研究强调Campomanes(2024)等权优于优化。但两者证据强度相当——IC加权在A股的证据同样来自国泰君安(2013, IR 2.38→2.63)和广发证券(2018, 超额+3.7%) | ⚠️ 中 |

### 1.2 需协调（方案层面对齐）

| # | 维度对 | 协调要点 |
|---|--------|---------|
| C3 | D1(VMP扩展到全部因子) vs D2(波动率倒数权重警告) | D1建议将波动率缩放扩展到全部5个因子（Wang & Li 2024），D2警告简单的波动率倒数权重会导致因子共线性。**解决方案**：VMP缩放 ≠ 波动率倒数权重。VMP缩放的是因子暴露（w = min(1.0, σ_target/σ_realized)），不是波动率倒数作为因子权重。可以安全扩展到全部因子。 |
| C4 | D3(科创板部分纳入) vs D5(行业集中度20%) | 科创板股票行业集中在电子/医药/计算机→如果纳入+行业集中度收紧到20%→科创板股票几乎肯定触达行业上限被替换。**解决方案**：科创板纳入为实验性（P2），v2.0暂不实施；若纳入，行业上限保持30%。 |
| C5 | D4(MA60→MA20) vs D4(非对称迟滞) | MA20比MA60更敏感+降仓恢复也加迟滞(3天)→可能导致更频繁的降仓/恢复切换。**解决方案**：先用MA20替代MA60观察效果，如果降仓频率过高再加回恢复迟滞。 |
| C6 | D4(50%降仓) vs D6(闲置现金管理) | 降仓后50K资金闲置+国偾逆回购年化~2%→~1000元/年≈组合的1%。**解决方案**：D6的现金管理是D4降仓的完美补充。 |

### 1.3 可共存（正向协同）

| # | 维度对 | 协同描述 |
|---|--------|---------|
| C7 | D1(EP→BP/复合估值) vs D2(IC加权) | 更好的估值因子(BP替代EP)→更高且更稳定的IC→IC加权可以更准确地给它更高权重。正向协同。 |
| C8 | D1(残差动量替代原始动量) vs D4(择时条件A改为MA20) | 残差动量对市场beta暴露更低→择时信号对策略影响更小→两者独立工作，不互相干扰。 |

## 二、证据收敛分析

以下结论得到**多维度独立研究的交叉支持**：

### 收敛发现 #1：因子权重不应严格等权

| 维度 | 发现 |
|------|------|
| D1 因子研究 | Wang & Li (2024): VMP对全部因子有效（夏普 0.99→1.50） |
| D2 信号合成 | 国泰君安(2013): IC加权 IR 2.38→2.63, 广发(2018): IC加权超额+3.7% |
| D4 风控研究 | 华泰金工(2026): 路径优选优于固定等权 |

**收敛强度**: ⭐⭐⭐⭐⭐（强收敛）
**含义**: v1.3的严格等权（动量除外）可以松绑。建议采用IC引导的权重 + VMP缩放作为权重上下限。

### 收敛发现 #2：BP优于EP作为价值因子

| 维度 | 发现 |
|------|------|
| D1 因子研究 | 华泰/山西/安信一致结论：BP的IC、单调性、多空收益均优于EP |
| D2 信号合成 | 复合估值因子(BP+EP+CFP) RankICIR 3.68 vs EP单独约2.0 |

**收敛强度**: ⭐⭐⭐⭐⭐（强收敛）
**含义**: EP→BP/复合估值是v2.0最高优先级变更。

### 收敛发现 #3：10只股票对10万资金是合理的

| 维度 | 发现 |
|------|------|
| D5 组合构建 | 8股更优（佣金门槛），但差异取决于是否免五 |
| D6 执行成本 | 免五不可行（监管收紧），维持10股为佳 |
| D3 可投资池 | 若纳入科创板→需要更多仓位来承载 |

**收敛强度**: ⭐⭐⭐⭐（强收敛，方向一致）
**含义**: 因为免五不可行（D6），8股的佣金优势不成立。**维持10股不变。**

### 收敛发现 #4：波动率管理是v2.0的核心框架改进

| 维度 | 发现 |
|------|------|
| D1 因子研究 | VMP扩展到全部因子（夏普翻倍级改善） |
| D4 风控研究 | 波动率过滤是择时信号最有效补充维度 |
| D2 信号合成 | VMP缩放优于简单波动率倒数作为因子权重 |

**收敛强度**: ⭐⭐⭐⭐⭐（强收敛）
**含义**: 波动率——作为单一风险维度——同时改善因子暴露管理(D1)、择时信号精度(D4)和权重合理性(D2)。应作为v2.0的核心设计原则。

### 收敛发现 #5：回测方法需要从单路径升级

| 维度 | 发现 |
|------|------|
| D7 数据与回测 | CPCV + PBO + DSR是2024年学术/业界标准 |
| D2 信号合成 | IC衰减需要严格滚动验证（中信建投2019） |
| D4 风控研究 | 参数网格搜索需要统计显著性检验 |

**收敛强度**: ⭐⭐⭐⭐（强收敛）
**含义**: v2.0应加入PBO和DSR作为回测验收标准。

## 五、最终方案架构 v2.0

### 因子体系

```
v1.3: EP + ROE + MOM(6-1M) + IV60 + Size (等权, 动量VMP)
v2.0: BP(+EP+CFP) + ROE + 残差MOM(6-1M) + IV60 + Size (IC引导 + 全因子VMP)
      可选: + 短期反转(1M)
```

### 信号合成

```
v1.3: Winsorize 1%/99% → Z-score → 等权(w_mom缩放) → Top10
v2.0: Log(Size) → 行业感知Winsorize 1%/99% → Z-score → IC衰减加权 × VMP全因子缩放 → Top10
```

### 风控体系

```
v1.3: MA60 + σ_20d>σ_median + 5日迟滞降仓 + 立即恢复
v2.0: MA20 + σ_20d>σ_median + 5日迟滞降仓 + 3日迟滞恢复 + 绝对回撤15%熔断
```

### 回测方法

```
v1.3: 2015-2021训练/2022-2025验证, 单路径
v2.0: 2010-2025数据, Rolling 3年训练/1年验证, CPCV + PBO + DSR验收
```

### 可投资池

```
v1.3: 7层过滤
v2.0: 10层过滤(+审计意见 +商誉/净资产>30% +质押≥50%)
```
