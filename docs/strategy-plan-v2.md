### 1.3 10 万资金交易成本

| 费用项 | 费率 | 收取方式 | 影响 |
|--------|------|---------|------|
| 佣金 | 万2.5，最低5元/笔 | 买卖双向 | 最大杀手 |
| 印花税 | 万5 | 仅卖出 | 次要 |
| 过户费 | 万0.1 | 仅沪市双向 | 可忽略 |

**成本测算（10 只股票，每只 1 万，月度换手 30%）：**

```
单只完整买卖：
  买入佣金：max(5元, 10000×0.00025) = 5元
  卖出佣金：max(5元, 10000×0.00025) = 5元
  卖出印花税：10000×0.0005 = 5元
  合计：15元

年化交易成本：15元 × 3只/月 × 12月 = 540元 ≈ 组合的 0.54%
```

> **[v2.0] 免五现状**: 2024-2025年监管收紧，免五（佣金免最低5元）基本绝迹。10只股票的持股数在免五不可得的情况下是合理的。

### [v2.0 新增] 闲置现金管理

当市场择时触发降仓（50%仓位），约5万元闲置资金：
- **国偾逆回购 GC001**：年化~1.5-2.5%，完全无风险，T+0可用（不影响调仓）
- 潜在年化增收：50,000 × 2% ≈ **1,000元 ≈ 组合的1%**

## 二、因子体系

### 2.1 [v2.0] 五个核心因子（重构）

| # | 因子 | 计算方式 | 数据来源 | 经济学解释 | 方向 | v2.0 变更 |
|---|------|---------|---------|-----------|------|----------|
| 1 | **价值 BP+** | BP(净资产/总市值)为主，EP+CFP为辅的复合估值 | `daily_basic.pb` (1/PB) | 便宜公司长期回报更高 | 越大越好 | **EP→BP+复合估值** |
| 2 | 质量 ROE | 年化 ROE | `fina_indicator.roe_yearly` | 高资本回报率公司持续创造价值 | 越大越好 | 维持不变 |
| 3 | **动量 残差MOM** | 过去6个月（排除最近1个月）市场+行业中性的**残差**累计收益 | `pro_bar` 复权收盘价 + 截面回归 | A 股残差动量有正溢价（原始动量是反转） | 越大越好 | **原始动量→残差动量** |
| 4 | 低波动 IV60 | 60 日日收益率标准差的倒数 | `pro_bar` 日收益 | A 股低波动异象 | 越大越好 | 维持不变 |
| 5 | 规模 Size | **ln(总市值)** 取负 | `daily_basic.total_mv` | 小盘长期溢价（权重最低） | 越大越好 | **log→ln变换** |

### [v2.0] BP+ 复合估值因子详细设计

```
═══════════════════════════════════════════════════════════
BP+ 复合估值因子
═══════════════════════════════════════════════════════════

来源：华泰证券(2016) + 山西证券(2024) + 安信证券(2024)
     三份独立研报一致结论：BP在A股的IC、单调性、多空收益均优于EP

三个子因子：
  BP  = 净资产 / 总市值 = 1 / PB
  EP  = 1 / PE_TTM（亏损股已排除）
  CFP = 经营活动现金流 / 总市值

复合方式（等权融合）：
  z_value = (z_bp + z_ep + z_cfp) / 3

其中：
  - CFP 数据来自 fina_indicator（经营活动现金流净额）
  - 每个子因子先独立截面标准化，再等权合成
  - BP天然对金融股无效（金融股PB<1不代表便宜）
    → 金融股（申万银行+非银金融）仅用EP+CFP两因子合成

RankICIR：3.68（安信证券实证）
多空年化：24.16%（vs EP单独约15-18%）

═══════════════════════════════════════════════════════════
```

### [v2.0] 残差动量因子详细设计

```
═══════════════════════════════════════════════════════════
残差动量因子
═══════════════════════════════════════════════════════════

核心问题：A股原始价格动量在短期(1-3月)是负的（反转效应）
         BigQuant(2024): 原始动量3月ICIR = -0.244（反转）
         残差动量3月ICIR = +0.15（正动量！）

来源：BigQuant(2024) + 华泰证券(2024-2025) + GF证券(2024)

计算步骤：
  Step 1: 对每个交易日t，跑横截面回归
          r_i(t-126, t-21) = α + β_mkt·r_mkt + Σβ_ind·I_ind + ε_i
          其中 r_i(t-126, t-21) = 股票i过去6个月（跳过最近1个月）的累计收益

  Step 2: 残差 ε_i = 原始动量 - 市场Beta部分 - 行业归属部分
          这就是"不能被市场和行业解释的、纯个股动量"

  Step 3: 残差动量 = ε_i（即回归残差本身）

预期效果：
  - 残差动量在A股呈现正动量溢价（与原始价格动量的反转效应相反）
  - 华泰证券：改进残差动量年化超额 12.90%
  - 对市场Beta暴露更低→择时信号变化时策略更稳定
  - 与原始动量因子的方向性改善（从可能拖累→正面贡献）

═══════════════════════════════════════════════════════════
```

### 2.2 因子选择原则

- 因子覆盖 Fama-French 五因子 + A股特有维度（反转/残差动量）
- 因子间相关性天然较低（价值与动量负相关，低波与规模对冲）
- **每个因子的计算方式经2023-2026最新A股文献独立验证**
- 每个因子可一句话解释"为什么存在"——不是数据挖掘产物

## 三、[v2.0] 信号合成流程

```
原始因子值
  │
  ├─ [v2.0] Size: ln(总市值) 变换（可选项，因子内处理）
  ├─ [v2.0] 行业感知去极值：Winsorize at 1%/99%（申万一级行业内部分别执行）
  │    目的：保留行业间结构性差异（如银行天然低PE，科技天然高PE）
  ├─ 截面标准化：Z-score = (x - μ) / σ
  ├─ 方向统一：确保"越大越好"
  │
  ├─ [v2.0 扩展] 全因子波动率自适应缩放（VMP）
  │    w_i = min(2.0, σ_target / σ_i_60d)  对每个因子i
  │    σ_target = 0.15（年化，Wang & Li 2024 A股最优参数）
  │    σ_i_60d = std(因子i的60日日收益) × √252
  │    上限2.0防止极端低波动时过度放大杠杆
  │    来源：Wang & Li (2024) + Barroso & Santa-Clara (2015) + Moreira & Muir (2017)
  │
  ├─ [v2.0 新增] IC衰减引导权重
  │    IC_weight_i = Σ_t decay(t) × RankIC_i(t)  / Σ_i Σ_t decay(t) × RankIC_i(t)
  │    回看窗口：24个月
  │    衰减函数：exponential decay with factor-specific half-life
  │    来源：国泰君安(2013) + 广发证券(2018) + 中信建投(2019)
  │
  ├─ 综合得分：
  │    Composite = Σ_i (IC_weight_i × w_VMP_i × z_i) / Σ_i (IC_weight_i × w_VMP_i)
  │    其中 w_VMP_i 为波动率缩放系数，IC_weight_i 为IC引导权重
  │    ≈ 三维修正等权合成
  │
  ▼
综合得分 → 选 Top 10 → 等权配置
```

**设计哲学**：v1.3的"严格等权"是防止过拟合的诚实做法。v2.0在保留这一核心原则的基础上，增加了两维有独立学术验证的修正：
1. **VMP缩放**（Wang & Li 2024）——当因子波动率异常时自动降低暴露
2. **IC衰减引导**（国泰君安2013 + 广发2018）——用历史IC信息温和调整权重

两项修正都不涉及优化——不估计协方差矩阵、不求解最优化问题——因此保留了"等权"的简洁性和抗过拟合能力。

### v2.0 综合得分公式

```
以前5因子为例：

Step 1: VMP缩放系数
  w_vmp_value  = min(2.0, 0.15 / σ_value_60d)   # BP+复合估值
  w_vmp_roe   = min(2.0, 0.15 / σ_roe_60d)
  w_vmp_mom   = min(2.0, 0.15 / σ_mom_60d)
  w_vmp_iv60  = min(2.0, 0.15 / σ_iv60_60d)
  w_vmp_size  = min(2.0, 0.15 / σ_size_60d)

Step 2: IC引导权重（24月滚动）
  IC_weight_i = normalized_decay_weighted_rank_IC_i

Step 3: 三维修正权重
  adj_weight_i = IC_weight_i × w_vmp_i

Step 4: 综合得分
  Composite = Σ(adj_weight_i × z_i) / Σ(adj_weight_i)

边界条件：
  - σ_i_60d → 0（停牌等）→ w_vmp_i = 2.0（上限，不缩放）
  - σ_i_60d → ∞ → w_vmp_i → 0
  - RankIC_i 转负 → IC_weight_i 仍参与归一化但设下限0.10（等权基线0.2的50%下限）
  - 所有IC_weight的均值为1/5（等权基线），偏离不超过±50%（即 [0.10, 0.30]）
```

### 参数总览

| 参数 | v1.3值 | v2.0值 | 来源 |
|------|--------|--------|------|
| σ_target（VMP目标波动率） | 0.15（仅动量） | 0.15（全部因子） | Wang & Li (2024) |
| VMP回看窗口 | 60交易日 | 60交易日 | Moreira & Muir (2017) |
| VMP上限 | min(1.0, ...) | min(2.0, ...) | 新增：允许适度放大 |

| IC权重上下限 | 无 | 等权±50% | 防过拟合安全阀 |

## 四、个股选择机制

### 4.1 第一步：构建可投资池（Universe）— [v2.0] 10层过滤

```
全市场 A 股 (~5000 只)
  │
  ├─ ① 排除 ST、*ST（退市风险警示股）
  │    → 剩 ~4800
  │
  ├─ ② 排除上市不满 1 年的次新股
  │    → 剩 ~4400
  │
  ├─ ③ [v2.0] 排除非CSI 300/500/1000成分股的科创板（688开头）
  │    原因：放宽原v1.3的完全排除。若科创板股票已是主要指数成分
  │          （如中芯国际、金山办公），则纳入
  │    约束：科创板持仓上限 = 2/10（20%），单只科创板仓位按200股/手计算
  │    → 剩 ~3950（比v1.3多约50只优质科技股）
  │
  ├─ ④ 排除 PE_TTM ≤ 0 的亏损股
  │    → 剩 ~2800-3200
  │
  ├─ ⑤ 排除近 20 日日均成交额 < 2000 万元的股票
  │    → 剩 ~1800-2200
  │
  ├─ ⑥ 排除股价 < 2 元的低价股
  │    → 剩 ~1750-2150
  │
  ├─ ⑦ 排除净资产 ≤ 0 的股票
  │    → 剩 ~1700-2100
  │
  ├─ ⑧ [v2.0 新增] 排除最近年度被出具非标审计意见的股票
  │    原因：2024年A股3.5%的公司收到非标意见（CSRC数据）
  │          非标意见是财务造假/退市的最强预警信号
  │    数据源：fina_indicator 或 v_audit 表
  │    → 剩 ~1650-2050
  │
  ├─ ⑨ [v2.0 新增] 排除商誉/净资产 > 30% 的股票
  │    原因：商誉减值风险——2024年全市场商誉减值638亿元（1,049家公司）
  │          高风险行业：医药、非银金融、电子
  │    数据源：fina_indicator.goodwill + fina_indicator.total_equity
  │    → 剩 ~1550-1950
  │
  └─ ⑩ [v2.0 新增] 排除股权质押比例 ≥ 50% 或控股股东质押 ≥ 80%
       原因：质押爆仓是A股闪崩的主要触发器
       数据源：pledge_stat API
       → 剩 ~1500-1900
```

**v2.0最终可投资池：约1300-2100只股票**（比v1.3的1500-2500更聚焦，尾部风险更低）。

### 4.2 第二步：逐股计算因子得分

```
以某股票为例：
  BP+   = 复合估值（BP + EP + CFP等权，金融股仅用EP+CFP）
  ROE   = roe_yearly              （来自 fina_indicator，年化口径）
  RES_MOM = ε_i（残差动量：原始6-1M动量对市场+行业回归的残差）
  IV60  = 1 / std(daily_returns, 60)   （60日波动率倒数）
  Size  = -ln(total_mv)           （[v2.0] ln变换后取负）
```

> **关键：财务数据必须用公告日（ann_date）而非财报截止日（end_date）** —— 同v1.3，保持不变。

### 4.3 第三步：截面标准化 [v2.0 增强]

```
[v2.0] Step A: 行业感知去极值（Winsorize）
  → 在申万一级行业内部分别执行 1%/99% 分位数截断
  → 目的：保留行业间结构性差异（银行天然低PB，科技天然高PB）
  → 行业内样本量不足30只时，回退到全市场去极值

Step B: Z-score 标准化
  → z = (x - 当日横截面均值) / 当日横截面标准差

Step C: 方向统一
  → 确保所有因子 "z 值越大 = 股票越好"
  → 本方案的5个因子天然同向，无需反转
```

### 4.4 第四步：三维修正等权合成 → 排名 → 选股

```
Step 1: VMP缩放系数计算（每个因子独立）
  w_vmp_i = min(2.0, 0.15 / σ_i_60d)

Step 2: IC衰减权重（24月滚动，每月更新）
  IC_weight_i = normalized_decay_weighted_rank_IC_i

Step 3: 调整后权重
  adj_weight_i = IC_weight_i × w_vmp_i

Step 4: 综合得分
  Composite = Σ(adj_weight_i × z_i) / Σ(adj_weight_i)

按综合得分从高到低排序
  │
  ├─ 初选 Top 10
  │
  ├─ 行业集中度检查（申万一级行业分类，31 个行业）
  │   └─ 如果某个行业在 Top 10 中 > 3 只（>30%）
  │       → 将该行业得分最低的股票替换为排名第 11、12... 的非同行业股票
  │
  └─ 最终 10 只 → 每只目标权重 10%
```

### 4.5-4.6 调仓执行、手数处理

与 v1.3 相同，保持不变。

## 五、[v2.0] 风险管理体系

### 5.1 三层风控架构

#### 第一层：组合层面（不变）

| 规则 | 参数 | 目的 |
|------|------|------|
| 单只股票最大权重 | 10% | 防止单只暴雷致命 |

#### 第二层：[v2.0] 市场择时叠加（参数优化）

```
[v2.0] 双条件触发（参数校准版）：

  条件A（趋势）：中证500收盘价 < MA20
       [v2.0 变更] MA60→MA20：长城证券(2016)证实中证500最优均线为20日
       原因：中证500波动率显著高于大盘指数，MA60过于迟钝

  条件B（波动率）：σ_20d > 过去60日σ中位数
      （维持v1.3设计）

  降仓信号 = 条件A AND 条件B AND 连续5个交易日确认
  → 仓位降至 50%（持有 5 只而非 10 只，其余现金或国债逆回购）

  [v2.0] 恢复满仓 = 条件A不成立 AND 连续3个交易日确认
       [v2.0 变更] 恢复加入3日迟滞（原为立即恢复）
       原因：华泰金工(2026)拐点识别率仅61.8%
             A股"牛短熊长"→立即恢复可能在下跌中继反弹时过早入场

  参数：
    ma_period = 20        # [v2.0] 均线周期（从MA60改为MA20）
    vol_window = 20       # 波动率计算窗口
    vol_percentile = 0.50 # 波动率阈值
    hysteresis_down = 5   # 降仓迟滞天数
    hysteresis_up = 3     # [v2.0 新增] 恢复迟滞天数

效果：MA20对中证500趋势反转响应更快，恢复迟滞防止假反弹
代价：MA20可能略微增加降仓频率（但双条件中的条件B已在振荡市过滤假信号）

学术来源：
  - 长城证券 (2016)：中证500最优均线为20日
  - 国信证券 (2018)：波动率过滤显著降低假信号
  - 华泰金工 (2026)：拐点识别率61.8%，恢复需确认
```

#### 第三层：[v2.0] 熔断机制（阈值收紧）

| 熔断条件 | 动作 | 恢复条件 | [v2.0] 变更 |

## 七、[v2.0] 防过拟合措施

1. **Walk-Forward 回测替代单次分割**：多个样本外路径→更可靠的泛化估计
2. **PBO + DSR 统计检验**：量化"策略是否可能只是幸运"
3. **因子数量控制**：5个核心 + 最多1个实验性 → 不变
4. **权重有安全阀**：IC权重偏离等权不超过±50%，VMP缩放上限2.0
5. **参数敏感性分析**：新增参数全网格搜索
6. **纸面交易验证**：不变
7. **实盘渐进入场**：30% → 60% → 100%

**v2.0核心原则**：在等权的"诚实"和优化的"效率"之间取平衡——用有独立学术验证的修正（VMP + IC引导），但每个修正都设安全阀防止过度拟合。

## 九、已知风险与局限（v2.0新增）

### 9.1 方法层面风险

| 风险 | 严重度 | 描述 | 缓解 |
|------|:------:|------|------|
| **IC权重过拟合** | ⚠️ 中 | 24月回看窗口在10年回测中仅有~120个点，IC估计可能不稳定 | ±50%安全阀；Phase 3 敏感性测试12/24/36月窗口 |
| **残差动量实施复杂度** | ⚠️ 低-中 | 截面回归需要正确的行业分类和足够的截面样本 | 使用SW2021 Level-1（31行业），每截面>1500股票，统计上充足 |
| **BP+金融股处理** | ⚠️ 低 | 金融股PB<1不代表低估（高杠杆行业的特性） | BP+中金融股仅用EP+CFP两因子，已编码 |
| **VMP上限2.0的校准** | ⚠️ 低 | 2.0上限来自Wang & Li(2024)参数分布分析，非最优解 | Phase 3敏感性测试1.5/2.0/2.5 |
| **BaoStock复权差异** | ⚠️ 低 | 涨跌幅复权法 vs 现金红利复权法存在系统性差异 | 仅用于交叉验证（比较日收益相关系数），不用于主策略 |

### 9.2 市场层面风险

| 风险 | 严重度 | 描述 |
|------|:------:|------|
| **MA20可能增加振荡市假信号** | ⚠️ 中 | MA20比MA60更敏感→趋势反转识别更快，但振荡市中可能更频繁触发条件A。双条件机制（需条件B同时成立）应过滤大部分假信号，但需回测验证 |
| **因子拥挤度上升** | ⚠️ 低-中 | Size因子拥挤度评分1.01（海通2024）。若量化私募继续扩张，BP/ROE等经典因子也可能拥挤。Phase 5监控 |

### 9.3 小资金特有局限

| 局限 | 描述 |
|------|------|
| **佣金5元最低** | 免五不可得（2025监管收紧）→每次1万交易=有效佣金万5。无法进一步降低成本 |
| **手数不可分割** | 100股=1手→权重偏差1-2%在统计上可接受但无法消除 |
| **流动性上限** | 10万资金不会遇到流动性瓶颈，但若未来资金增长到50万+，2000万ADV阈值需要上调 |
